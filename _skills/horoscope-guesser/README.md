# Horoscope Guesser Skill

A fun and interactive Markdown-only skill that asks 5 personalized questions and generates a customized horoscope reading based on your answers.

## Features

- **5 Interactive Questions:** Asks about mood, lucky color, lucky number, hopes, and personality
- **Personalized Horoscope:** Generates unique readings based on your answers
- **Lucky Elements:** Provides lucky numbers and colors
- **Personality Insights:** Reflects your personality in the reading
- **Daily Predictions:** Includes customized predictions and advice
- **No Code Required:** Markdown-only implementation for easy interpretation by AI

## Files

- `SKILL.md` - Skill metadata, instructions, and implementation
- `README.md` - This file

## Skill Type

**Markdown-Only Skill** — Pure instructions for AI assistant, no code implementation needed.

## Usage

Invoke via the gaia-agent-skills framework:
```bash
/horoscope-guesser
```

The AI assistant will:
1. Greet you warmly
2. Ask 5 questions about your mood, lucky color, number, hopes, and personality
3. Generate a personalized horoscope reading combining all your answers

## How It Works

The SKILL.md file contains:
- Complete instructions for the AI assistant
- 5 specific questions to ask
- Guidance on creating personalized horoscopes
- A prediction bank for variety
- Format templates for consistent output

The AI assistant reads these instructions and follows them to conduct the interactive horoscope reading experience.

## Customization

You can customize the experience by editing SKILL.md:
- Change the 5 questions
- Add more predictions to the prediction bank
- Adjust the mystical tone or emoji usage
- Modify the horoscope format

## Notes

- This is a pure Markdown skill with no code implementation
- The AI assistant interprets the instructions and conducts the interaction
- Each horoscope is unique based on user answers
- The experience is entertainment-focused and lighthearted
